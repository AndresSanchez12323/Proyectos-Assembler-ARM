# Proyectos de Arquitectura de Hardware (ARMv6)

Este repositorio agrupa ejercicios en ensamblador ARMv6. Cada proyecto tiene su propia carpeta y README con la explicacion del codigo.

---

## Proyectos

- Batalla Naval: ver `batalla_naval/README.md`
- Ahorcado: ver `ahorcado/README.md`
- Serie de Fibonacci: ver `serie_fibonacci/README.md`

---

## Estructura

- `batalla_naval/`: juego por turnos con tableros 8x8.
- `ahorcado/`: juego de adivinanza de palabras con limite de intentos.
- `serie_fibonacci/`: calculo iterativo de Fibonacci.

---

## Notas generales

- El codigo esta pensado para ARMv6 y entornos basicos.
- Los comandos de compilacion pueden variar segun el toolchain.
- Para pruebas se uso QEMU emulando un entorno Raspberry Pi.
